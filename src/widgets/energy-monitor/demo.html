Конструктор віджетів <energy-monitor>

Додавайте, видаляйте й налаштовуйте віджети через форму — праворуч живий перегляд. Кожен віджет незалежний. Кнопка «Зберегти configs.json» завантажує налаштування; далі node generate.mjs створить для кожного віджета окрему папку з готовим кодом.

📦 Генерація окремих папок: збережіть configs.json у цю теку (widgets/energy-monitor/), потім виконайте node generate.mjs — зʼявиться dist/<slug>/ для кожного віджета з index.html + energy-monitor.js. Таку папку можна копіювати на будь-який сайт.
