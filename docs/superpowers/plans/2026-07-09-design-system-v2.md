# Design System v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Modernize the ses-ai-studio design system to a light-first emerald + solar-gold system (Sora + Manrope, tokenized), and apply it to 4 flagship sections — changing only presentation, never functionality/content/structure.

**Architecture:** Rebuild the token layer in `src/index.css` (`@theme` colors + `:root` scales + `.ds-*` component classes). Then de-hardcode the 4 flagship components via an exhaustive, deterministic color-string migration map plus three per-file edits (display-font headings, dark/light surface via `.ds-*`, viewport-fit tuning). The `energy-monitor` widget is already on the target palette and is not touched.

**Tech Stack:** React 19, Tailwind CSS v4 (`@tailwindcss/vite`, CSS-first `@theme`), Vite 6, TypeScript, lucide-react, motion. Fonts via Google Fonts `<link>` in `index.html`.

## Global Constraints

- **Branch:** all work on `feature/design-system-v2` (already created; spec committed).
- **Presentation-only.** Do NOT change: component logic, `useApp`/AppContext, localStorage, data flow, i18n keys or copy, DOM structure, section count/order, `onClick`/`openLeadModal`/`addLead` calls, form field `name`/`value`/`onChange`, honeypot, `energy-monitor`, AI widget, calculator, shop, CRM, routing. Edit only `className` strings, `src/index.css`, `index.html` fonts, and `docs/DESIGN-SYSTEM.md`.
- **Section fits one viewport.** Each main section's content fits ~650–700px usable height; no scroll *within* a section. Verify at **1366×768** and **1440×900**: top heading AND bottom CTA visible without in-section scroll.
- **No test runner exists.** Per-task verification = `npm run lint` (tsc --noEmit) passes + `npm run build` succeeds + browser visual/viewport check. There are no unit tests to write.
- **Contrast floor:** body ≥ 4.5:1, large text ≥ 3:1 on every background.
- **Never touch the monitor's attribute values.** In `HeroSection.tsx` the strings `powers-home="5,10,30"` and `powers-biz="30,50,100"` are power presets, NOT colors — exclude from every replacement.
- Spec: `docs/superpowers/specs/2026-07-09-design-system-v2-design.md`.
